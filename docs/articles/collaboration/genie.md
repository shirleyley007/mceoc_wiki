# Genie

Genie is a Teams bot which integrates with MCEOC data, allowing you to quickly query **special shift members** such as today's on-call engineer or afternoon shift (AS) engineer.

### Example Queries

| Query | Result |
|-------|--------|
| `oncall for vm+scim today` | Shows VM and SCIM pod on-call engineers with role (Primary/Standby) and shift type |
| `AS for net today` | Shows NET pod afternoon shift engineer |
